USING SEEKER - EXAMPLES

1) OVERVIEW 

ev

In this section we show a number of examples that illustrate how to use Seeker and also some of its capability. 

We will begin with input because it is very similar for both Correlation and Factor Analysis. In both you specify the target that you want to analyze; it can be a stock, index, ETF, or a portfolio. It can also be a group: if you select Stock, for example the group fills with stock groups that you can select from (including All stocks in the system). Alternatively, for small comparisons, you can type a comma-delimited set of tickers in the lower window. If you use this, Seeker will override any group selection in the upper window; so if you are processing a group, you need to ensure that the lower window is cleared.

The match to be compared against is specified in the same way,

We then need a timescale over which to compare them, and we show the earliest date for which we have data (when you are working with two portfolios you are not using our data and so analysis can start earlier, if you have the data). In the timescale we provide subintervals; the reason for this is to break up the correlation into periods so that you can see how it veried during the overall timeperiod. We output statistical data to give you a sense of how consistent the  subintervals are.

Finally, because the output can be large, we give options for you to pick the number of rows that you want to see, whether you want it sorted in ascending or descending order, and the column on which you want to sort.

For analysis you have 4 methods to choose from: 

Period Analysis correlates over the timescale and subintervals that you choose.

Phase Analysis does not use subintervals, it moves the data of the Match forward in time in relation to the Target by up to 2 years. If correlation improves, there is the possibility that the Match is leading the price of the Target and the two should be carefully examined in Period Analysis.

Autocorrelation uses the same approach as Phase Analysis, but uses the Target as the Natch. If shifting the Target in time in relation to itself improves correlation, its price is probably cyclical.

The final method, Portfolio style, analyzes a Target portfolio against all our Indexes over time, showing how its correlation with industry groups changes over time. 

Finally, after any of the first 3 analyses, you can click Alpha or Beta to see the correlation coefficient replaced by the alpha or beta over each period, to answer the question "correlation is all well and good, but how did they perform. Both alpha and beta are normalized to show the ratio of Target/Match.

7    INDEX AND ETF NAMES

Sometimes users want to clarify the groups and need help identifying an index or an ETF. When you run a Seeker analysis the right hand margin shows two pull-downs, one for indexes, and one for ETF's. If you click either, a drop-down show an alphabetic list of their symbols and full names. 

If you need to know which stocks are included in a stock or index group, select Autocorrelation, and the group name as a target. Run the analysis with any date, and the result will list constituent symbols.

2) PHASE ANALYSIS

Phase Analysis does not use sub-intervals. Output is currently a grid, with the sort based on MaxNonZeroValue. Charting options will be added.

The columns show the Target and Match tickers, and their types. On the right are shown the correlations for different phase offsets. For example, SPNC/DJVG had a correlation coefficient of 0.91 when they were analyzed at the same time, but when DJV6 is advanced by 21 trading days (1 month) the correlation coefficient rises to 0.96. NOTE that each of the examples on the page shows the pairs with the highest correlation out of several thousand, This shows less variability than appears lower down the tables! Consequently a difference of 5% for SPNC/DJVG is unlikely to be worth evaluating, but if the gap is larger, it may be worth inserting the pair into a Period Analysis to examine them further, or to set up the analysis over a longerperiod to see if it becomes stronger. The grid is currently sorted by the offset that gives the highest correlation.


3) Autocorrelation

The example correlates Industrials with all indexes through 2015. The output is identical to Phase Analysis, except that when an entity is correlated against itself, the correlation coefficient must be 1.0 at zero offset, as you can see. In autocorrelation we are looking for cyclical behavior, which would be seen if correlation increased or decreased with the phase advance. As you can see, CR appears to be in the top row, IO's correlation with itself falls from 0.98 to 0.02 at 315 days(15 months) and rises to 0.93 at 441 days. As with the phase offset, correlating intervals can be analyzed further with Period Analysis, or by extending the time scale to 10 years.


Autocorrelation has two additional features: (1) if you want to see the members of a group, autocorrelating the group will list its members. (2) Autocorrelation lists the full name of an entity alongside its ticker; if you are unsure of a symbol name autocorrelating it will show the full hane.


Portfolio Style Analysis

Seeker's portfolio style analysis measure how correlation between a portfolio and external index averages changes over time.

Charts automatically present all the data on the grid, so there is no row selection. The grid output can be exported to Excel. Note that the portfolio illustrated only has data through 2014. 

 

This analysis selects the Value indexes, Large-, Mid-, and Small-Cap. What it indicates for this portfolio is that it correlates better with Large Caps, in its last full year, 2014, it correlated at 0.7 against the average of the Large, and 0.5 against the average of the Small stocks, In 2011, it did not correlate well with any index group 


This output uses the Growth and Value indexes. You can see that correlation is mixed from year to year.

Here is how the portfolio performed against group averages for our 20 sector index groups. It can be seen that correlation has been much lower from 2011. 

 Note that in order to create index groups, Seeker analyzes index names and groups them accordingly. This leads to some indexes being included in more than one group; for example, an S&P Automotive index would be included in both Automotive and S&P groups. As these are analyzed in different charts, there is no overlap.

Here is the comparison with the average of 11 proprietary index groups. Correlation is highest with Wilshire, Value Line and Russell. As we saw with the sector groups, correlation has been lower since 2011.. 

 



The outputs that you have seen will evolve over time, and we welcome user participation in suggesting improvements.
