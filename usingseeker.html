USING SEEKER - EXAMPLES

1) OVERVIEW

Originally developed for a hedge fund, Seeker uses correlation to find relationships between entities that include individual and grouped stocks, indexes, portfolios and EFTs.

If, for example, the Energy sector is correlated with all 1,500 indexes available in the system, the result is 150,000 sorted pairs of correlations, depending on the period analyzed. It takes seconds to analyze.

Correlation is primarily a tool for finding linear relationships: correlation does not prove causality, but as Seeker correlates on closing prices, finding that the prices of two entities follow each other with 90%+ correlation is remarkable, and you will find many that do. Our “Technical” tab discusses the significance of correlation coefficients.

Sector analysis identifies a corporation with a single group identity that is usually based on its history. Because correlation is based on actual performance, it offers an alternative to sector analysis. We show two examples below:

At the end of section (1) Period Analysis, we show how correlating a sector with itself exposes wide disparities in performance. 

the second, in (4) Portfolio Style Analysis, shows how correlating with stock sector and index groups can provide an analysis of style change over time in relation to 1,500 indexes, 10 sector groups and 38 index groups.

Seeker’s main features are data integrity and performance. Data integrity enables it to handle source data that may include input errors and gaps. Speed enables makes it very responsive, which increases analyst productivity, and to analyze entire industries.

The standard system contains 2,900 stocks, 1,500 indexes, 1,300 ETF's, 10 stock sector groups, and 38 index groups that include index owners, like “Wilshire”; types, like “Growth”, industry and others.

Seeker users work in their own data space; they can upload their own stock and index groups, watch lists, and portfolios, and can even base their correlation on different data sets, if required.

2) CORRELATION OPTIONS

    1) Period Analysis

Seeker's first option is period analysis. A “Target” is defined that can be selected from a dropdown list that is displayed when you enter a stock, ETF, index or portfolio or, it can be entered as a single symbol, or a comma-delimited string of symbols. Seeker gives preference to manually entered items, so if there are some displayed, they override the dropdown selection. A “Match” option to compare with the Target is similarly defined. Seeker pairs, and analyzes, every possible combination of Target and Match entered, which can comprise thousands of combinations. 

 A timescale is defined that can cover any period within the Earliest and Latest Dates. Seeker calculates the overall correlation, but also lets you select Sub-intervals, which allow you to evaluate granularity over shorter periods. Finally, the results are sorted, and you can select an ascending sort (for positive correlation) or descending (for negative correlation) and choose the column that you want to sort. Data entry: if you target "Stocks", for example, a drop down list displays a list of sector groups. Users can build and upload their own watchlists that are displayed as part of this group. Or, users can type in a comma delimited list of stock, or index tickers, which is convenient for doing one-off comparisons. When the correlation analysis is complete the user can select alpha (ratio of the interval gain in closing price gain for the Target divided by that of the Matches) , or beta (ratio of the interval standard deviations of the Target, divided by that of the Matches. The results are shown in the same order in which they were correlated to make comparison easier.




    2)Phase Analysis

Seeker correlates the Target and Match over the designated period, then repeats the correlation by advancing the Match in time for periods from 5 days to 2 years. Phase offsets are measured in days. If, for example, a stock is correlated with an index with a phase difference of 20, the most recent stock is paired with the index data from 20 days ago, etc. If correlation improves, the index appears to be leading (or predicting) the stock. If you find a pair of interest, run a Period Analysis to check them in more detail.

    3) Autocorrelation 

This is similar to Phase Analysis except that the correlation is run by comparing the Target with itself. With zero phase offset the correlation coefficient must be 1.0. If correlation improves at some phase offset, the stock, index or portfolio is exhibiting cyclical behavior. If you find a pair of interest, enter their symbols and run a Period Analysis to see more detail.

    4) Portfolio Style Analysis

Load an existing portfolio into Seeker.Select it as the Target and choose a correlation timescale, broken down into sub-intervals. The analysis compares all 1,500 indexes with the portfolio performance over each subinterval, averaging the results by index group. The results show how the performance of the portfolio has compared to different indexes over time, reflecting the change in “style”.

3) OUTPUT

1) Period Analysis

The example is the case used to demonstrate input, a correlation of the Healthcare sector against all 1,500 indexes through 2015, using monthly sub-intervals. The Period Analysis output is a grid, showing each pair of correlated entities at the left, with their type. Matches are often indexes, and as their symbols are more cryptic, we included at the right-hand side of the screen a reference pull-down that contains all the indexes in the system (we also list the ETF's). The grid is sorted on Overall correlation but any of the columns may be selected.

 

 


The top row is the average of the analysis. The 168 Healthcare stocks and 1,500 indexes yielded 250,000 active results during 2015. They are sorted on Overall correlation. The number of rows to be displayed, and the column to sort, can be selected.

The leftmost columns show a "Select" column that is used to select the row for charting. The "Overall" data column shows the overall correlation over the entire period, and then the results are shown for 12 monthly sub-intervals, with their average, median, high and low values, followed by the result for each interval.

Once the analysis is run, you have two choices, select a row and chart it, or you can do an Alpha or Beta analysis on the grid. To look at the charts, Select a row in the grid, then click "Select, then Chart".

You will occasionally find a cell with "NaN" (not a number) which is Seeker's way of indicating that it could not calculate a value for that interval.

The 168 Healthcare stocks and 1,500 indexes yielded 250,000 active results during 2015. They are sorted on Overall correlation. The number of rows to be displayed, and the column to sort, can be selected.

This second example of Period Analysis shows the Barclays group of ETF's compared against all Indexes. The top line shows EBND correlated with @BKLA-I at 98% overall; we also show the pulldown for Indexes which shows that BKLA-I is the Bony Latin America Adr Index. (A second pulldown shows the full names of the ETF's available.



A third example of Period Analysis shows a portfolio selected as the Target and All Stocks as the Match. This correlates the performance of the portfolio with all 2,900 stocks currently in the system. The purpose is to look for new stocks that performed similarly to the portfolio. The selection could have been limited to "stocks in a particular sector", for example, or the data could have been sorted "Overall Ascending" which would have shown those stocks that behaved least like the portfolio and might therefore be used as shorts. 

 The comparison is over 2014 (since data for our sample portfolio ends early in 2015). It shows that more than 50 stocks correlated at over 70%, with a high of 77%. If you want to know what level of correlation is significant in trending, see the discussion on the "Technical" tab. 

  


If we go back to the first example, showing Healthcare, and click on “Alpha” ratio, you will see the following grid, redrawn with correlation values replaced by the ratio of the Target gain over that of the Match for each period. The gain for each is the ratio of the value at the end of the interval divided by the value at the start of the interval. If the grid shows an interval value of 1.5, that implies that the Target gained 50% more than the Match over the period.

If you click on “Beta” ratio, the grid is redrawn with correlation values replaced by the ratio of the Target standard deviation over that of the Match for each period. As standard deviations reflect price, they are normalized by the average price over the period. If the grid shows an interval value of 2, for example, that implies that in that period, the Target was twice as volatile as the Match, compared to their underlying prices. 

[NOTE] Standard deviation is often referred to as "volatility". This can be misleading because the standard deviation knows nothing about the trend, so a growth stock that climbs steadily can have the same standard deviation as one that steadily falls, or another that simply fluctuates about a constant value. A stock that climbs steadily will be scored as volatile, even though it may have steadily grown in value every week!] 

 Any of the grids in Seeker can be output to Excel (complete! They are not limited by the number of rows selected for display).

HOW WELL DO SECTORS CORRELATE?

The following charts show the Healthcare sector correlated with itself during 2015. Stocks in a sector are assumed to perform similarly, and the first chart shows that there are many that matched very well. Note that Seeker automatically prevents autocorrelation because it confuses group comparison, and is provided as a separate Seeker mode.


But when the correlation grid is sorted in ascending order, it shows that many of the stocks in the sector had correlation of -0.9 indicating that the match did almost the opposite of the target. It shows that you cannot assume that sector stocks can be relied upon to produce similar performance. The overall average across the sector was only 0.15! 

2) PHASE ANALYSIS

Phase Analysis does not use sub-intervals. Output is currently a grid, with the sort based on MaxNonZeroValue. Charting options will be added.

The columns show the Target and Match tickers, and their types. On the right are shown the correlations for different phase offsets. For example, SPNC/DJVG had a correlation coefficient of 0.91 when they were analyzed at the same time, but when DJV6 is advanced by 21 trading days (1 month) the correlation coefficient rises to 0.96. NOTE that each of the examples on the page shows the pairs with the highest correlation out of several thousand, This shows less variability than appears lower down the tables! Consequently a difference of 5% for SPNC/DJVG is unlikely to be worth evaluating, but if the gap is larger, it may be worth inserting the pair into a Period Analysis to examine them further, or to set up the analysis over a longerperiod to see if it becomes stronger. The grid is currently sorted by the offset that gives the highest correlation.


3) Autocorrelation

The example correlates Industrials with all indexes through 2015. The output is identical to Phase Analysis, except that when an entity is correlated against itself, the correlation coefficient must be 1.0 at zero offset, as you can see. In autocorrelation we are looking for cyclical behavior, which would be seen if correlation increased or decreased with the phase advance. As you can see, CR appears to be in the top row, IO's correlation with itself falls from 0.98 to 0.02 at 315 days(15 months) and rises to 0.93 at 441 days. As with the phase offset, correlating intervals can be analyzed further with Period Analysis, or by extending the time scale to 10 years.


Autocorrelation has two additional features: (1) if you want to see the members of a group, autocorrelating the group will list its members. (2) Autocorrelation lists the full name of an entity alongside its ticker; if you are unsure of a symbol name autocorrelating it will show the full hane.


Portfolio Style Analysis

Seeker's portfolio style analysis measure how correlation between a portfolio and external index averages changes over time.

Charts automatically present all the data on the grid, so there is no row selection. The grid output can be exported to Excel. Note that the portfolio illustrated only has data through 2014. 

 

This analysis selects the Value indexes, Large-, Mid-, and Small-Cap. What it indicates for this portfolio is that it correlates better with Large Caps, in its last full year, 2014, it correlated at 0.7 against the average of the Large, and 0.5 against the average of the Small stocks, In 2011, it did not correlate well with any index group 


This output uses the Growth and Value indexes. You can see that correlation is mixed from year to year.

Here is how the portfolio performed against group averages for our 20 sector index groups. It can be seen that correlation has been much lower from 2011. 

 Note that in order to create index groups, Seeker analyzes index names and groups them accordingly. This leads to some indexes being included in more than one group; for example, an S&P Automotive index would be included in both Automotive and S&P groups. As these are analyzed in different charts, there is no overlap.

Here is the comparison with the average of 11 proprietary index groups. Correlation is highest with Wilshire, Value Line and Russell. As we saw with the sector groups, correlation has been lower since 2011..
